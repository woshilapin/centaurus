centaurus
=========

The relativist ray-tracer

# Prerequisite

You should install a bunch of dependencies to compile `centaurus`:
* autoconf (> 2.68)
* automake (> 1.9)
* autopoint (> 0.18.1)
* libboost-dev
* libmagick++-dev
* [optional] doxygen
* [optional] libboost-test-dev
